# Python Server Swarm

Python Server Swarm is a containerized multi-bot chat system with distinct personalities, social dynamics, and optional local LLM integration.

## Highlights

- Multiple bot personas with different behaviors
- Realtime chat over Socket.IO
- Docker-first setup for the server and swarm workers
- Optional LM Studio or local model endpoint integration

## Quick start

```bash
make up
```

Useful commands:

```bash
make status
make logs
make down
```

Default local URL: `http://localhost:5000`
